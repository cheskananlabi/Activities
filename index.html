About

This webpage serves as a compilation of my activities for the first and second semesters. It contains the different projects and exercises I completed in each quarter as part of my web development learning experience. These activities helped me understand the basics of creating webpages using HTML and applying simple design concepts through layout and structure.

Throughout each quarter, I practiced building webpages that focus on proper formatting, organization of content, and visual presentation. The works included in this compilation show my improvement over time, from simple page designs to more organized and creative layouts. Each activity represents the effort I put into learning and applying the lessons discussed in class.

This compilation not only documents my progress but also reflects my growing interest in web development. By exploring each section, you can see how my skills developed across the semesters and how I gradually became more confident in creating functional and presentable webpages.
